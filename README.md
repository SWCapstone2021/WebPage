## 찾아봐유 홈페이지

이 프로젝트는 2021 Ajou University Spring SW capston design 과목의 일환으로 진행되었습니다.

해당 레포는 찾아봐유의 홈페이지 소스코드를 저장하고 있습니다. 



## 주요 기능 소개

홈페이지에서는 다음과 같은 기능을 제공 합니다.

-[ ] 메인 페이지
-[ ] 인증 기능
    - [ ] 로그인
